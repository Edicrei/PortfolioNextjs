## About the project

The website was created in Next.js. The content is loaded from JSON files. I also used add-ons such as: Framer Motion, NodeMailer, React Hook Form, React Icon, tsParticles and Typed.js.

## Technologies Used

- HTML
- Bootstrap
- TypeScript
- React.js
- Next.js
- Styled Components
- JSON
- API
